CSS ONLY COMPONENTS

In this repository you will find components using only HTML and CSS such as accordions, tooltips and modals.

Feel free to fork this project, modify the existing components or add a new one.

What you will find:

  • ACCORDION
  • MODAL
  • TOOLTIPS

What you will need:

NODE

Install:

Get the project:

$ git clone https://gitlab.fftech.info/Labs/css-only.git

Open terminal in the directory of the project and run

$ npm install

Have fun!

Notes:

If you want to make changes in the SASS files, please run the following code in the terminal to update your css files
$ gulp
